<b>Beschreibung:</b>

Addon zum exportieren vorhandener MySQL Tabellen ins XLS-Format.
Es besteht die Möglichkeit nur die "neuen" Datensätze seit des letzten Exports zu exportieren.

Ausserdem ist der zuletzt erstellte Export als Download verfügbar.

<b>Installation:</b>

- Unter "redaxo/include/addons" einen Ordner "excel_export" anlegen

- Alle Dateien des Archivs nach "redaxo/include/addons/excel_export" entpacken

- Den Ordner "redaxo/include/addons/excel_export/files" mit Schreibrechten versehen.

- Im Redaxo AddOn Manager das Plugin installieren

- Im Redaxo AddOn Manager das Plugin aktivieren

- Dem Benutzer das recht "excel_export[]" verleihen

- fertig ;)
