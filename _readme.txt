// ---------------------- INSTALLATION

1. entpacken

1.1. redaxo.zip oder redaxo.tar.gz entpacken
1.2. sie sollten einen ordner mit htdocs_redaxo erhalten

2. upload der daten

2.1. laden sie den inhalt des htdocs_redaxo ordners in ihr DocumentRoot verzeichnis (z.b. htdocs oder www oder www.servername.de) über z.b. ftp.
2.2. in ihrem DocumentRoot verzeichnis muesste nun eine "index.php" datei und der "redaxo" ordner liegen.
2.3. geben sie in ihren browser ihre serveradresse ein. www.servername.de
2.4. sie sollten "Kein Startartikel selektiert" erhalten. wenn nicht dann nochmal bei 2. anfangen

3. setup

3.1. rufen sie www.servername.de/redaxo/setup.php auf und führen sie die dort angegebenen schritte durch

4. wenn alles erfolgreich abgeschlossen

4.1. löschen sie /redaxo/setup.php oder verschieben sie diese datei.
